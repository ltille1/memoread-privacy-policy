Politique de confidentialité – MemoRead Journal

MemoRead Journal respecte votre vie privée. Cette politique explique quelles données sont collectées, comment elles sont utilisées et quels sont vos droits.

Données collectées

1. Adresse e-mail

Votre adresse e-mail est collectée via Firebase Authentication afin de :

  • créer et gérer votre compte utilisateur ;
  • vous permettre de vous connecter à l’application ;
  • sécuriser l’accès à vos données personnelles de lecture.

2. Contenu utilisateur (stocké dans Firebase Firestore)

L’application enregistre le contenu que vous ajoutez, notamment :

  • références de livres ;
  • notes de lecture ;
  • tags, catégories, commentaires ou autres métadonnées que vous choisissez d’ajouter.

Ces données sont stockées de manière sécurisée dans Firebase Firestore et sont associées à votre compte afin de permettre la sauvegarde et la synchronisation entre vos appareils.

Utilisation des données

Les données collectées sont utilisées uniquement pour :

  • fournir les fonctionnalités principales de l’app (authentification, stockage, synchronisation) ;
  • afficher et organiser vos notes et références de livres ;
  • assurer le bon fonctionnement technique du service.

Ce que nous ne faisons pas

  • Nous ne vendons pas vos données.
  • Nous n’utilisons pas vos données pour de la publicité ciblée.
  • Nous ne faisons pas de suivi entre sites ou entre applications (pas de cross-app tracking).
  • Nous ne partageons pas vos données avec des tiers à des fins commerciales.

Services tiers

MemoRead Journal utilise les services Google Firebase (notamment Authentication et Firestore) pour la gestion des comptes et le stockage des données. Pour plus d’informations sur la manière dont Firebase traite les données, vous pouvez consulter :
https://firebase.google.com/support/privacy

Durée de conservation des données

Vos données (compte, notes, références de livres) sont conservées tant que votre compte reste actif. Si vous demandez la suppression de votre compte, les données associées sont supprimées de manière définitive dans un délai raisonnable.

Suppression du compte et des données

Vous pouvez demander la suppression de votre compte et de toutes les données associées en nous contactant via la page suivante :
https://github.com/TON_COMPTE_GITHUB/memoread-privacy-policy/issues

Contact

Pour toute question relative à la confidentialité ou à vos données, vous pouvez nous contacter via :
https://github.com/TON_COMPTE_GITHUB/memoread-privacy-policy/issues
